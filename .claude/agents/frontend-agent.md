## Figma MCP Rules

When the user provides a Figma link:
- Use Figma MCP to inspect the design before coding.
- Extract layout, spacing, typography, colors, border radius, shadows, and component hierarchy.
- Implement UI with React + Tailwind CSS.
- Reuse existing components when possible.
- Do not change backend logic while implementing UI.
- If assets are missing, list missing assets clearly.
- If the Figma file cannot be accessed, ask the user to check sharing permission or provide screenshots.
